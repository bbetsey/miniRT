# miniRT | 107/100
## Mini Ray Tracing with miniLibX

**Build it**
```
make
./miniRT [filename]
```

**Figures**
* plane
* triangle
* square
* sphere
* cylinder

**Syntax**
```
R [width] [height]                                                      // window resolution
A [rate] [r,g,b]                                                        // ambient lighting
c [x,y,z]*coordinates* [x,y,z]*normalized orientation vector* [r,g,b]   // camera
```
